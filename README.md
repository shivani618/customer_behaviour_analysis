# Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow, starting from data loading and exploration to visualization and reporting. The objective is to transform raw data into meaningful business insights using industry-standard tools and techniques.

The project includes:

* Loading the dataset using Python
* Performing Exploratory Data Analysis (EDA)
* Cleaning and preprocessing the data
* Writing SQL queries in PostgreSQL
* Building an interactive Power BI dashboard
* Creating a business insights report
* Designing a presentation using Gamma

---

## Dataset

The dataset contains customer purchase and shopping information, including customer details, purchased products, categories, discounts, payment methods, ratings, and purchase behavior.

Key columns include:

* Customer ID
* Item Purchased
* Category
* Purchase Amount
* Discount Applied
* Review Rating
* Payment Method
* Season
* Gender
* Location

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Verified data types and dataset structure.
* Checked the number of rows and columns.

### 2. Exploratory Data Analysis (EDA)

* Analyzed customer purchasing patterns.
* Explored product categories.
* Identified trends in sales and discounts.
* Visualized distributions and relationships using charts.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data.
* Standardized text values.
* Prepared the dataset for analysis.

### 4. SQL Analysis (PostgreSQL)

Performed SQL queries to answer business questions such as:

* Top-selling products
* Most popular categories
* Average ratings
* Discount analysis
* Customer purchase behavior
* Ranking products using window functions
* Aggregate and grouped analysis

### 5. Power BI Dashboard

Created an interactive dashboard featuring:

* Sales overview
* Category performance
* Customer insights
* Discount analysis
* Review ratings
* Filters and slicers for dynamic analysis

### 6. Business Report

Prepared a report summarizing:

* Key findings
* Business insights
* Customer trends
* Recommendations based on the analysis

### 7. Presentation

Created a professional presentation using Gamma to communicate the project's objectives, methodology, insights, and conclusions.

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook and run the data loading, cleaning, and EDA steps.
4. Import the cleaned dataset into PostgreSQL.
5. Execute the SQL queries.
6. Open the Power BI (.pbix) file to explore the dashboard.
7. Review the project report and presentation for detailed insights.
